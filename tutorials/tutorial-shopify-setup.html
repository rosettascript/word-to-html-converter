Advanced Setup Process

  1. Prepare your Word content — Use proper headings (H1-H6), create lists with Word's list tools, format Key Takeaways sections clearly
  2. Open the converter — Navigate to the main page and use the sidebar toolbar for easy access to all options
  3. Paste and preview — Paste your content and see it rendered in the input area (not raw HTML code)
  4. Choose the right Shopify mode — Select Shopify Blogs for articles/stories or Shopify Shoppables for product descriptions
  5. Configure advanced options — Enable strong tags in headers (default ON), convert internal links to relative paths, control paragraph spacers
  6. Review and preview — Toggle between code view and rendered preview to verify the output
  7. Copy and test — Copy the clean HTML and paste into Shopify. Test on mobile and desktop
  8. Optimize if needed — Adjust optional features and re-convert if the output needs tweaking
