## Quick Reference

**Phase Timeline:**
- **Phase 1 (Alpha):** Week 1-2 - Internal testing
- **Phase 2 (Closed Beta):** Week 3-5 - Selected users
- **Phase 3 (Open Beta):** Week 6-8 - Public beta
- **Phase 4 (GA):** Week 9+ - Full launch

**Key Deliverables:**
- Web UI with paste functionality
- Three output modes (Regular, Shopify Blogs, Shopify Shoppables)
- Instant conversion (no button)
- Preview toggle (code view ↔ rendered view)
- SEO-optimized page structure
- JavaScript library (P1)
