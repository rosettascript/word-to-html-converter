# AXE Accessibility Reports (archived)

The automated accessibility scanner outputs (axe-core) were generated during development and auditing.
To keep the repository lean, the full JSON reports have been removed from the root and summarized here.

Files removed from repository root:

- `axe-results.json` — initial scan output (large JSON)
- `axe-results-after.json` — intermediate scan output
- `axe-results-final.json` — final scan output
- `axe-results-final2.json` — final verification run

If you need the original JSON reports, they can be regenerated by running the axe CLI locally:

```bash
npx --yes @axe-core/cli "file:///path/to/index.html" -s axe-results.json
```

Note: The automated scans were used to drive accessibility fixes; the final scan reported zero automated violations after iterative fixes.
